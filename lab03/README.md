Це Python-проєкт, що демонструє метод стеганографії LSB (Least Significant Bit) для приховування текстових повідомлень у звичайних зображеннях.
Програма дозволяє кодувати текст у зображення та витягувати його назад без втрати даних.

Можливості:

* приховування тексту у зображення методом LSB;
* використання трьох каналів (R, G, B) для збільшення ємності;
* додавання спеціального маркера `###END###` для визначення кінця прихованого повідомлення;
* витягування повідомлення назад у текст;
* автоматичне створення тестового зображення;
* демонстрація коректності методу на реальних персональних даних;
* аналіз змін між оригінальним і стего-зображенням.

Алгоритм роботи

**Приховування**

1. Конвертувати текст у байти, а потім у біти.
2. Додати спеціальний маркер завершення (`###END###`).
3. Відкрити зображення та отримати доступ до пікселів.
4. Для кожного біта повідомлення:
   * взяти піксель;
   * замінити молодший біт (LSB) каналів R, G, B;
   * зберегти оновлений піксель.
5. Зберегти модифіковане зображення у форматі PNG (без втрат).

**Витягування**

1. Відкрити стего-зображення.
2. Зчитати всі молодші біти каналів R, G, B.
3. Перетворити зібрані біти назад у текст.
4. Зупинитися при знаходженні `###END###`.
